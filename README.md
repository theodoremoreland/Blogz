# Blogz
A Python-Flask web application where multiple users can create their own Blog (updated to use Flask blueprints and Bootstrap CSS).  

*This was originally a homework assignment for LaunchCode's Lc101 (2018)*

## Technologies Used
Python (Flask)
Bootstrap
HTML/CSS
MySQL
